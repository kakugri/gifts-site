Update log
April 22, 2026

Runnable reference package added to the prototype page

The prototype materials now include a downloadable public reference package so readers can review the runnable package, not only the report PDF and entry-point file. The package now also includes stable input and output contracts, a CloudTrail-like adaptation example, sample output artifacts, and a short end-to-end walkthrough from raw events to suspicious-session interpretation. The prototype page was also clarified to present GIFTS as a reference implementation for review and adaptation.

April 21, 2026

Initial public site launch

The public GIFTS site launched with the core overview, technology, publications, prototype, and contact pages, along with linked public materials for technical review.

April 21, 2026

Content and metadata clarification

The site was updated with clearer author attribution, stronger publication details, improved prototype review guidance, and richer page metadata for public sharing and indexing.

April 21, 2026

Updates page and public site identity assets

An updates page, last-updated signals, site icon, and public share card were added so visitors can better identify the project and see when the materials were refreshed.

April 21, 2026

Page clarity and layout polish

Internal page layouts were tightened and page copy was refined so the public site reads more naturally for first-time visitors.
